## Overview

`libfwupd` is a stable C API and ABI client library used by GUI applications, command-line tools, and scripts to communicate with the fwupd daemon over D-Bus. It provides objects that mirror the daemon's data model — devices, releases, remotes, security attributes, and BIOS settings — plus an async client for all daemon operations.

### Main header

```c
#include <fwupd.h>
```

This single header pulls in every public type, including `FwupdClient`, `FwupdDevice`, `FwupdRelease`, `FwupdRemote`, `FwupdSecurityAttr`, and `FwupdBiosSetting`.

## Async API pattern

All daemon operations follow the GLib async pattern: call the `_async` variant with a `GAsyncReadyCallback`, then call the matching `_finish` function inside the callback to retrieve the result or `GError`.

```c
static void
on_devices_ready(GObject *source, GAsyncResult *res, gpointer user_data)
{
    FwupdClient *client = FWUPD_CLIENT(source);
    g_autoptr(GError) error = NULL;
    g_autoptr(GPtrArray) devices = fwupd_client_get_devices_finish(client, res, &error);
    if (devices == NULL) {
        g_printerr("Error: %s\n", error->message);
        return;
    }
    for (guint i = 0; i < devices->len; i++) {
        FwupdDevice *dev = g_ptr_array_index(devices, i);
        g_print("%s: %s\n", fwupd_device_get_name(dev), fwupd_device_get_version(dev));
    }
}
```

## Example: connecting and listing devices

```c
#include <fwupd.h>

static void
on_connected(GObject *source, GAsyncResult *res, gpointer user_data)
{
    FwupdClient *client = FWUPD_CLIENT(source);
    GMainLoop *loop = user_data;
    g_autoptr(GError) error = NULL;

    if (!fwupd_client_connect_finish(client, res, &error)) {
        g_printerr("Failed to connect: %s\n", error->message);
        g_main_loop_quit(loop);
        return;
    }
    g_print("Connected to fwupd %s\n", fwupd_client_get_daemon_version(client));

    /* kick off a device listing */
    fwupd_client_get_devices_async(client, NULL, on_devices_ready, loop);
}

static void
on_devices_ready(GObject *source, GAsyncResult *res, gpointer user_data)
{
    FwupdClient *client = FWUPD_CLIENT(source);
    GMainLoop *loop = user_data;
    g_autoptr(GError) error = NULL;
    g_autoptr(GPtrArray) devices = fwupd_client_get_devices_finish(client, res, &error);

    if (devices == NULL) {
        g_printerr("Error: %s\n", error->message);
    } else {
        for (guint i = 0; i < devices->len; i++) {
            FwupdDevice *dev = g_ptr_array_index(devices, i);
            g_print("  %s (%s)\n",
                    fwupd_device_get_name(dev),
                    fwupd_device_get_version(dev));
        }
    }
    g_main_loop_quit(loop);
}

int
main(void)
{
    g_autoptr(GMainLoop) loop = g_main_loop_new(NULL, FALSE);
    g_autoptr(FwupdClient) client = fwupd_client_new();

    fwupd_client_connect_async(client, NULL, on_connected, loop);
    g_main_loop_run(loop);
    return 0;
}
```

## Migration notes

### From 1.9.x to 2.x

| Old API | New API |
|---|---|
| `fwupd_build_machine_id()` | `fwupd_client_get_host_machine_id()` |
| `fwupd_build_history_report_json()` | `fwupd_client_build_report_history()` |
| `fwupd_get_os_release()` | `fwupd_client_get_report_metadata()` |
| `fwupd_device_get_update_message()` | Removed — drop use |
| `fwupd_device_get_update_image()` | Removed — drop use |
| `FWUPD_DEVICE_FLAG_ONLY_OFFLINE` | Removed — drop use |

Note: `fwupd_client_emulation_load()` and `fwupd_client_emulation_save()` now take a file path rather than returning or accepting bytes.

### From 1.x to 2.x

| Old API | New API |
|---|---|
| `fwupd_client_install_release2()` | `fwupd_client_install_release()` |
| `fwupd_client_refresh_remote2()` | `fwupd_client_refresh_remote()` |
| `fwupd_remote_get_enabled()` | `fwupd_remote_has_flag()` |
| `fwupd_remote_get_approval_required()` | `fwupd_remote_has_flag()` |
| `fwupd_device_get_protocol()` | `fwupd_device_get_protocols()` |
